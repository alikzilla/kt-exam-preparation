# КТ Подготовка — Магистратура М094 (ИКТ)

Веб-приложение для подготовки к комплексному тестированию (КТ) при поступлении
в **профильную магистратуру, группа М094 (Информационно-коммуникационные
технологии)**. Интерфейс на русском языке.

## Формат экзамена (официальный)

Профильный тест КТ для М094:

| Параметр | Значение |
| --- | --- |
| Всего вопросов | **50** |
| Дисциплина 1 — Алгоритмы и структуры данных | 30 вопросов |
| Дисциплина 2 — Базы данных | 20 вопросов |
| Продолжительность | **100 минут** (1 ч 40 мин) |
| Пороговый балл по каждой дисциплине | **не менее 7** |
| Формат вопроса | один правильный ответ из вариантов (А–Д) |

Конфигурация формата вынесена в [`src/data/exam.ts`](src/data/exam.ts).

## Возможности

- **Экзамен** — полный формат (50 вопросов, таймер 100 минут, авто-сдача по
  истечении времени), вердикт **Сдал / Не сдал** по порогу каждой дисциплины.
- **Тренировка** — выбор дисциплин и числа вопросов, опциональный таймер.
- Перемешивание вопросов **и** вариантов ответов при каждой попытке.
- Навигатор по вопросам, разбор ответов (верно/неверно, правильный ответ,
  пояснение).
- История попыток сохраняется в браузере (localStorage).

## Запуск

```bash
npm install --registry https://registry.npmjs.org   # корпоративный реестр блокирует часть пакетов
npm run dev      # дев-сервер, открыть напечатанный адрес localhost
npm run build    # проверка типов + сборка
```

## Замена демо-вопросов реальными бланками

Сейчас вопросы — демонстрационные. Всё приложение читает банк вопросов через
**один модуль**, поэтому замена реальными бланками КТ не затрагивает интерфейс и
логику.

1. Отредактируйте JSON-файлы в [`src/data/questions/`](src/data/questions/) —
   по одному на дисциплину. Формат вопроса описан типом `Question` в
   [`src/types/index.ts`](src/types/index.ts):
   - `correctOptionIds` — массив: один id для одного правильного ответа,
     несколько — для «выберите все подходящие».
   - `imageUrl` и `explanation` — необязательные.
2. Дисциплины (названия, число вопросов) — в
   [`src/data/subjects.ts`](src/data/subjects.ts) и
   [`src/data/exam.ts`](src/data/exam.ts).
3. Новый файл дисциплины зарегистрируйте в
   [`src/data/index.ts`](src/data/index.ts).
